## Задача 1: Волшебный калькулятор (Функция с return)
Объяснение: Ты создашь функцию, которая умеет складывать два любых числа и возвращает результат, чтобы его можно было использовать дальше.

### Задание:

Напиши функцию сумма(a, b).

Внутри функции вычисли результат сложения a и b.

Используй return, чтобы отдать результат наружу из функции.

Вызови функцию с числами 5 и 3, сохрани результат в переменную ответ и напечатай его.

### Пример решения:

```python
def сумма(a, b):
    результат = a + b
    return результат # Возвращаем результат наружу

ответ = сумма(5, 3) # Функция ВОЗВРАЩАЕТ число 8, и мы его ловим в переменную
print("Ответ:", ответ) # Напечатает: Ответ: 8

# Можно сразу напечатать то, что вернула функция
print("10 + 15 =", сумма(10, 15))
```

## Задача 2: Кухня робота-повара (Несколько функций с return)

Объяснение: Робот-повар выполняет рецепт. Каждый шаг рецепта — это отдельная функция, которая что-то возвращает.

### Задание:

Напиши функцию взять_яйца(количество). Она должна вернуть строку: "Взял(а) [количество] яиц."

- Напиши функцию взбить(продукт). Она должна вернуть строку: "Взбил(а) [продукт]."

- Напиши основную функцию приготовить_омлет(), которая:

- Вызовет взять_яйца(3) и сохранит результат в переменную.

- Вызовет взбить("яйца") и сохранит результат в переменную.

- Вернет строку: "Готово! Вот твой омлет!"

- Вызови приготовить_омлет() и напечатай, что она вернула.

Пример решения:

```python
def взять_яйца(количество):
    return f"Взял(а) {количество} яиц." # Возвращаем готовую фразу

def взбить(продукт):
    return f"Взбил(а) {продукт}."

def приготовить_омлет():
    шаг1 = взять_яйца(3)  # Сохраняем то, что ВЕРНУЛА функция
    шаг2 = взбить("яйца") # Сохраняем то, что ВЕРНУЛА функция
    print(шаг1) # Печатаем результат шага
    print(шаг2)
    return "Готово! Вот твой омлет!" # Финальный результат рецепта

итог = приготовить_омлет()
print(итог)
```
## Задача 3: Магический зверь (Класс с методами, которые возвращают значения)
Объяснение: Мы создадим волшебного питомца. У него будут методы (действия), которые не просто печатают, но и возвращают информацию о нём.

### Задание:

- Создай класс ВолшебныйЗверь.

- В конструкторе __init__ задай имя зверя и его уровень магической силы (сила).

- Создай метод представиться(), который возвращает строку: "Привет! Я [имя]!"

- Создай метод использовать_силу(), который возвращает строку: "[имя] использует силу уровня [сила]! ⚡"

- Создай зверя, вызови его методы и напечатай то, что они возвращают.

Пример решения:

```python
class ВолшебныйЗверь:
    def __init__(self, имя, сила):
        self.имя = имя
        self.сила = сила

    def представиться(self):
        # Возвращаем приветствие, а не печатаем его
        return f"Привет! Я {self.имя}!"

    def использовать_силу(self):
        # Возвращаем описание действия с силой
        return f"{self.имя} использует силу уровня {self.сила}! ⚡"

# Создаем зверя
мой_зверь = ВолшебныйЗверь("Огнегрив", 10)

# Методы ВОЗВРАЩАЮТ строки. Мы их ловим и печатаем.
приветствие = мой_зверь.представиться()
print(приветствие)

атака = мой_зверь.использовать_силу()
print(атака)
```

## Задача 4: Секретное послание (Функция с return и условием)
Объяснение: Шифровальная машина получает код и возвращает расшифрованное слово, но только если код правильный.

### Задание:

- Напиши функцию расшифровать(код).

- Внутри функции сделай так:

- Если код равен 123, верни слово "ДРУГ".

- Если код равен 456, верни слово "МИР".

- Если код другой, верни фразу "Неизвестный код!".

Попробуй расшифровать коды 123, 456 и 999. Напечатай, что вернула функция для каждого кода.

Пример решения:

```python
def расшифровать(код):
    if код == 123:
        return "ДРУГ"       # Возвращаем это слово, если код подошел
    elif код == 456:
        return "МИР"        # Возвращаем другое слово
    else:
        return "Неизвестный код!" # Возвращаем, если код не совпал

# Пробуем разные коды
print("Код 123:", расшифровать(123))
print("Код 456:", расшифровать(456))
print("Код 999:", расшифровать(999))
```

## Задача 5: Счетчик игроков (Класс, который возвращает информацию о состоянии)
Объяснение: Мы создадим класс для командной игры. Этот класс будет следить за количеством игроков и возвращать важную информацию.

### Задание:

- Создай класс Команда.

- В конструкторе задай название команды и создай пустой список игроки.

- Создай метод добавить_игрока(имя), который добавляет имя в список.

 - Создай метод количество_игроков(), который возвращает длину списка игроков (используй len()).

- Создай метод сила_команды(), который возвращает число: количество игроков, умноженное на 10 (каждый игрок дает 10 единиц силы).

- Протестируй: создай команду, добавь 2-3 игроков и выведи на экран, что возвращают методы о количестве игроков и силе команды.

Пример решения:

```python
class Команда:
    def __init__(self, название):
        self.название = название
        self.игроки = [] # Пока никого в команде

    def добавить_игрока(self, имя):
        self.игроки.append(имя)

    def количество_игроков(self):
        # Возвращаем ЧИСЛО - длину списка
        return len(self.игроки)

    def сила_команды(self):
        # Сила зависит от количества игроков. Возвращаем ЧИСЛО.
        return self.количество_игроков() * 10

# Создаем команду
команда_а = Команда("Молния")

# Добавляем игроков
команда_а.добавить_игрока("Саша")
команда_а.добавить_игрока("Маша")

# Используем методы, которые ВОЗВРАЩАЮТ значения
сколько = команда_а.количество_игроков()
сила = команда_а.сила_команды()

print(f"В команде '{команда_а.название}' {сколько} игрока.")
print(f"Общая сила команды: {сила}.")
```
### Ключевые моменты для объяснения ребенку:

return — это как ответ на вопрос. Функцию можно спросить: *"Сколько будет 5+3?"*, а она возвращает ответ: "8".

Без return функция может что-то делать (печатать), но не оставляет после себя результата, который можно сохранить в переменную или использовать в других вычислениях.

В классах методы с return позволяют «спрашивать» у объекта информацию о нем (как его зовут, сколько у него силы и т.д.).
