# Папка с поэтапными уроками

Здесь шесть простых версий проекта (по одному файлу `main.py` в каждой папке). Каждая версия — маленький шаг для учеников 10–12 лет. Все уроки содержат подробные комментарии, объясняющие каждый шаг.

## Структура уроков:

- **`lesson_01_window`** — только окно и линия земли. Базовый запуск Pygame.

- **`lesson_02_move_jump`** — добавляем игрока, движение влево/вправо и прыжок с гравитацией.

- **`lesson_03a_shoot`** — учимся стрелять в сторону мышки. Подробно объясняется вычисление направления, векторы и нормализация.

- **`lesson_03b_projectile_physics`** — добавляем физику снарядов: рикошет от стен, падение с гравитацией, ограничение количества снарядов (максимум 3) и их подбор.

- **`lesson_04a_enemy_spawn`** — создаём врагов, которые ходят к игроку и появляются по таймеру. Учимся работать со списками и таймерами.

- **`lesson_04b_enemy_interaction`** — добавляем взаимодействие: снаряды уничтожают врагов, столкновение с врагом отнимает жизни, игра заканчивается при потере всех жизней.

- **`lesson_05_input_output`** — изучаем ввод данных от пользователя, условные операторы и перебор кортежей. Практические задания на интерактивные программы.

## Как использовать:

Можно показывать каждую папку как отдельный урок: сначала изучаем код с комментариями, потом запускаем (`python main.py`) и наблюдаем результат. Каждый урок строится на предыдущем, постепенно усложняя игру.
